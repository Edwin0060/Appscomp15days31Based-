Public Holidays

This module is used to define public holidays for the year

Configure Public holidays.

Using one click announce public holidays to employees via email notification.

Detailed public holidays report.

Configuration is available in Time Off > Configuration > Public Holidays

Define date range and public holidays details here.


Click on button Announce public holidays to share with employees,

From Print option can print report.
