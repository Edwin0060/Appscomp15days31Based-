SCREENSHOTS

SMS to a Single Customer

The user can send SMS to a single customer by selecting the Contacts app and using the SMS button located in front of each customer in the list view. User can also send SMS by clicking on Send SMS button inside individual contact form. To send sms, user must follow the E.164 number format i.e. start the mobile number with '+' followed by country code.



SMS to Multiple Customer

The user can send SMS to multiple customers at a time. The Bulk SMS tab can be located under SMS Gateway main menu. In the send sms form the user must follow the E.164 number format i.e. start the mobile number with '+' followed by country code. The user need to select gateway account from select account field. The user can have multiple accounts with the same SMS Gateway Provider.

Create Multiple SMS Groups and Send SMS

The user can create multiple group with each group can have multiple contacts in it. While sending the Bulk SMS, user can select "Use Customer Group" option and can send SMS to members in those group.



Sent SMS History

SMS history is maintained for both Single customer and Multiple customers. To get single SMS history, click on the Single SMS History tab and to get Bulk SMS history, click on Bulk SMS History tab from SMS Gateway main menu.



Resend Already Sent SMS

The user can resend the sent SMS to the customer to whom the message has been already sent by selecting the entry of that sms from the history (Single SMS or Bulk SMS History).

Create and Use Predefined Template

The user can create and use the predefined templates while sending the SMS and can make changes to those templates according to the requirement.



Auto Send SMS to Customer on their Birthday

Create single birthday template for all customers/ contacts. Inside Contacts app, add date of birth to a Contact to whom user wish to send birthday wishes. SMS will be sent automatically to a Contact on his birthday with template that has been defined already.



USER GUIDE

SMS Notifications Module Installation & Configuration Guide

Step 1: Enable SMS feature to specific user as per the requirement
To use SMS Notifications module, admin must allow 'Enable SMS Feature' to different user. SMS Gateway menu will not be visible if 'Enable SMS Feature' is deactivated for user.



Step 2: Create and Login to Twilio account
To use the twilio gateway, user need to create a twilio Account first.


Step 3: Get Account SID, Authentication Token and Twilio Number
After login to twilio using given credentials get Account SID, Authentication Token and Twilio Number from twilio dashboard.



Step 4: Add customer mobile numbers as a verified numbers in twilio dashboard and only then user can send messages from SMS Notifications module



Step 5: User need to install twilio on their system
Command :- sudo apt-get install python3-twilio


Step 6: Create SMS Gateway from SMS Notifications Module
Using SID, Authentication Token and Twilio Number, user can now create SMS gateway inside SMS Notifications module. To do this go to the 'SMS Gateway' tab from SMS Gateway main menu and fill required fields.
