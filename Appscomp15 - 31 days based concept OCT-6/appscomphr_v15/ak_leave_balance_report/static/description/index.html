Time Off Utilization Report

A summary of your employees' time off allocations, balances with a utilization rate for each time off type

About the App

The report is a list view with a summary of all time off allocations, approved requests as well as the balance days grouped by time off type. A utilization rate is also provided against each time off type to highlight how much is each employee taking advantage of their leave balances. The list view provides options to filter and search through data as needed. Unlike Odoo's time off analysis report, this report groups all transactions of a every time off type under one category.

The report is compatible with and has been tested on Odoo's Community and Enterprise versions.
